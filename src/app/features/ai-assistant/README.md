# AI Assistant Module

AI 助理模組 - Google Generative AI 驅動的聊天助理

## Structure

- `pages/` - AI assistant page component
- `features/` - AI-specific features (store, etc.)
- `data-access/` - AI repository and types
- `services/` - AI business logic services
- `routing/` - Route configuration

## Routes

- `/ai-assistant` - Main AI assistant page

## Features

- AI-powered chat interface
- Context-aware responses
- Conversation history
- Model selection
