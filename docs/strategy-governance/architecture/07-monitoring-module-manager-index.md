# 監控與模組管理索引 (Monitoring Module Manager Index)

> 來源：`docs-old/architecture/MONITORING_MODULE_MANAGER_INDEX.md`、`*_ANALYSIS.md`、`*_SOLUTION.md`、`*_VISUAL_GUIDE.md`。彙總監控/模組管理的脈絡與方案。

## 範圍
- 指標/告警設計、模組間責任分工、可視化/導覽、方案比較。

## 內容映射
- **INDEX**：導覽與主題列表
- **ANALYSIS**：現況與缺口、風險
- **SOLUTION**：建議架構與落地步驟
- **VISUAL_GUIDE**：圖示化導覽
- **EXECUTIVE_SUMMARY**：已另存於 operations/06（監控摘要）

## 行動重點
- 指標與告警預設集；告警節流與去重
- Blueprint 邊界：監控資料需含 blueprintId 過濾
- Runbook 對齊：告警事件對應 incident/rollback/security 手冊
- 定期演練與審查：效能、成本、配額
